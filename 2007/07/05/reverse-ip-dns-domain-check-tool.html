Lesezeichen: Reverse IP DNS Domain Check Tool

Lesezeichen: http://www.seologs.com/ip-domains.html

Bei großen Hostern, liegt der eigene Webspace ja immer auf einem Servern mit Haufenweise Kram von anderen Leuten. Wer immer schonmal wissen wollte, wer diese anderen Leute / Domains sind, kann das hier nachschauen lassen.

Lustige Bonusinfos:
a) Meine .ws Domains, die ja defintiv auf dem selben Server liegen müssten, sind nicht dabei. Sehr gut!
b) Andersherum geht es aber: .ws Domain eingeben, alle anderen (nich .ws) Domains auf dem Server sehen.
c) Auf meinem Server liegen noch 263 weitere Domains. Darunter so Kleinode und Goldgruben wie

herz-kaese.de
spittl-narr.de
militaerbestaende.de

nachmorgen.de
gangwerk.de
klangkoerperstimme.de

binsenweise.de

Das ist wirklich bizarr, das rauszufinden. Eingentlich ist es ja egal. Jede Seite im Internet ist von jeder anderen genau gleichweit, nämlich nur einen Klick entfernt. Aber die Vorstellung, dass "anmut und demut" auf einem Server mit "Gangwerk - Praxis für Hundephysiotherapie" liegt, dass meine Bilder und Dateien direkt neben Militärbeständen liegen - das fühlt sich schon komisch an.
