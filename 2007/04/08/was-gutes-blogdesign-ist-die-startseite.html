Was gutes Blogdesign ist - Die Startseite

Die Startseite ist die Zweitwichtigste. Sie ist mit Sicherheit die meistbesuchte Seite. Sie muss ein Fest sein. Sie hat eigentlich ewas wirklich verschwenderisch Schönes und Großes verdient, wie es bswp. Sylvia Loh vormacht. Gleichzeitig soll sie möglichst viele Angebote machen ohne unübersichtlich zu wirken. Die wenigsten letzten 5-10 Einträge sollte deutlich auffindbar sein und chronologisch geordnet. Während die normale Blogstartseite völlig automatisch aus den letzten Beiträgen, den letzten Kommentaren usw. generiert wird, werden die angebotenen Artikel bei Online Zeitungen von Hand gesetzt. Die Startseite von Blogs jedes Mal komplett von Hand zu gestalten ist zu viel Arbeit, aber was man immer häufiger sieht sind einzelne Artikel, die von Hand an einen bestimmten Platz gesetzt werden. So kann man die Aufmerksamkeit auf besonders Gute Artikel lenken, die sonst zu schnell von der Startseite verschwunden wären. Auch hierfür bietet das Standard-Blogsystem keine Lösung.

Ressorts

Außederm sollten die 5-7 wichtigsten Ressorts auf die Startseite. Überhaupt finde ich es eigentlich ziemlich wichtig, dass ein Blog seine Inhalte nicht nur einfach über einen Haufen Kategorien oder einen noch größeren Haufen Tags koordiniert, sondern über ein kleines, mit Bedacht gewähltes Vokabular von Ressorts. Diese Ressorts würde ich auch auf jeder Seite entweder im Header, einer Sidebar oder dem Footer deutlich machen. Die Ressorts sollten dem Leser einen guten Überblick über die Themen geben, die im Blog abgehandelt werden.

Der Autor

Auf die Startseite gehört außerdem ein bischen was zum Autor. Blogs sind Personen. Das Blog unterscheidet sich von der Zeitung durch seine Menschlichkeit. Ein Foto, ein Statement, ein Motto, ein Name und sei es nur wie hier ein Pseudonym.

Nach Draußen

Eine Blogroll ist eine faire Sache und ich habe schon zu machen guten Blog über die Blogroll anderer Blogs gefunden. Auch sie sollte einen festen Platz auf der Startseite haben. Entweder vollständig oder nur als Verweis auf eine extra Blogroll seite. Nun bin ich allerdings nicht der Meinung, dass die auf jeder Seite in der Sidebar aufzutauchen hat.

Inzwischen werden in Blogs auch gerne Randnotizen oder Kurzlinks angeboten. Das sind in der Regeln entweder sehr kurze Blogeinträge mit einem Link auf eine andere, externe Seite, oder resyndizierte RSS-Feeds von del.icio.us oder die Shared Items aus dem Google-Feed-Reader oder etwas ähnlichem. In aller Regeln handelt es ich um Links nach Draußen, raus aus dem Blog. auch sie gehören auf die Startseite, zusammen mit Flickr-Feeds, sofern vorhanden. Schon auf den anderen Übersichtsseiten machen sie nur noch wenig Sinn und auf den Artikelseiten erst Recht nicht.

Der Fuchs in Tüten.

Die Startseite muss Spaß machen, immer wieder. Und Sie sollte beindrucken. Sie sollte das sein, was Dieter Rams als "innovativ" bezeichnet hat. Sie muss sich abheben von anderen Startseiten. Ein Konzept muss her, das ist klar.

So kann man zum Register auch ein Lexikon aufbauen. Gute Idee?
