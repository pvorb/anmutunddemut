Wordpress 2.3 avec die Tags oben drauf

Wordpress 2.3 wird mit einer eigenen Tag-Enginge und einer runderneuerten Taxonomie-Term-Topic-Verwaltung daher kommen. Damit kommt Wordpress von Haus aus mit einem guten Teil der Funktionalitäten daher, die ich in meinem Semantic Weblog Plugin habe, und obendrein schneller und schöner. Ich kann ein bischen verstehen, warum sie dafür eine eigenen Termverwaltung gebaut haben, aber ich verstehe nicht, warum sie nicht ihr eigenes Metadatensystem verwenden. Das ist so mächtig, eigentlich. Naja, abwarten.

Damnit.
