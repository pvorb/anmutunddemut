Pieksende Krümel

Webprogrammierung ohne Encodingprobleme ist wie Frühstück im Bett ohne Krümel. Inwzischen hab' ich diverse Manntage mit dem Beheben von Encoding Problemen verbracht. Das fiese an der Sache ist, dass man in PHP einem String ja auch nicht so ohne weiteres ansehen kann wie er encodiert ist. Mit ein bischen Gefussel kann man das allerdings doch hinbekommen:

if(mb_detect_encoding($string, 'UTF-8, ISO-8859-1')!="UTF-8")
	$string = utf8_encode($string);
}
