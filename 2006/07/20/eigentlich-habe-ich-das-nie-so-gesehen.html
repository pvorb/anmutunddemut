Eigentlich habe ich das nie so gesehen

Kaum lungert man eine Weile in Hamburg rum, schon wird man zu Austellungseröffnungen eingeladen. Das ist nett. Ausgesprochen nett. So stellt man sich als Bielefelder Hamburg vor: Die Zeit, der Spiegel und Ausstellungseröffnungen. Im Linda gibt es morgen abend "Eigentlich habe ich das nie so gesehen", eine Gruppenausstellung zum Thema "Postkarten". Das ist schon mal ein schöne Idee. Überhaupt sind ja schöne Ideen das Schönste an der ganzen Kunst.
