Typo3 on Birkenhake.org

I got my own Typo3 running on birkenhake.org. the installation was quite easy, but without an introduction, I could manage to create a new page on my empty project. Eike had to do this for me.

My first success: I took a prepared template and filled it with my content. My first defeat: It's very tricky to change something in the template, if you dont't know how...

And... very scary... the template.html contains: tables, tables tables, font-elements, "1px x 1px"-images and it is not wellformed at all. Huhu, the ghosts of the dead and gone are still alive typo3.

I wonder if it's possible to create valid AAA XHTML 1.1 with Typo3....
