Smarty Templates

Smarty Templates sind ein tolle Sache, um PHP-Code von HTML Code zu trennen: Statt einer PHP-Datei macht man eine Template Datei mit HTML und lustigem Kram in runden Klammern drin und noch eine Datei mit dem PHP Code drin, und einem Smarty-Onjekt. Dann verweist man aus der PHP Datei auf das template und schon geht's los...

Und damit ich den Kombis mal ziegen kann, dass das auch prima läuft, wird Buddhabot jetzt auf Smarty umgebaut. Zu dem Zweck hab ich eben mein erstes Template und meine erste smarty.php auf meinem neuen Server zum Laufen gebracht.

War gar nicht schwierig.
