Smarty CSS PHP - Trio mit vier Fäusten

Für HDnet hab ich eine lustige kleine Webseite gebastelt, so eine Datenbank mit Artikeln, Bsuchern, Benutzern und Administratoren; mit Workflow und Medienverwaltung. Alles drum und dran. HTML PHP, Javascript, CSS MySQL; die ganze Palette.

Und weil dir Projektplanung und Projektentwicklung eine gute Eigendynamik hatte, mach ich jetzt den ganzen Spass noch mal neu: Objektorientiert, mit Smarty und massiver Ausnutzung der @import Funktion von CSS und meiner Idee vom perfekteb CSS-Netzwerk dank css.php. Kurz: ich mache all das richtig, was ich meinen Studenten jede Woche predige.
