Das MS Imperium schlägt zurück

Microsoft kann man viel vorwerfen, aber nicht, dass sie keine Ahnung vom Geschäft haben. Die aktuelle Office Suite hat einen niegelnagelneuen massiven XML Support, was nur konsequent ist, da das .NET-Framework krass XML zentriert ist. Also haben die Jungs nd Mädels bei Microsoft ihre Standart-Office-WErkzeuge hergenommen um XML-Editoren draus gemacht: jeder nach seinen Fähigkeiten, jedem nach seinen Bedürfnissen.

Dabei geht Mircosoft deutlich weiter als die OpenSource Konkurrenz OpenOffice, die "nur" ein eigenes XML Format haben und alle OO-Dokumente nativ darin speichern. MS Office, kann nun XML-Dokumente jeglichen Formats lesen, als XML bearbeiten und wieder speichern:

Excel eigenet sich naturgemäßt gut für tabellarische Daten und hat zudem eine Schnittstelle zu XML-Schema:

via heise "
Schnelle Tabelle
"

Word hat naturgemäß einen Editor für Mixed-Content, der stark an den XMetal erinnert:

via heise
"What you see is what you tag"

Fasst man die neue XML-Funktionalität der Office-Suite mit der XML-Orientierung von .Net und dem kostenlosen .NET webspace zusammen, so bleibt einem nur, den Hut vor Microsoft zu ziehen. Respekt.
