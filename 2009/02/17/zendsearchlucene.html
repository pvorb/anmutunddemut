@Anonymous:

Durch die Kapselung von Lucene durch den Apache Solr Server kannst Du Lucene in der Tat überall verwenden. Die Suchanfragen werden dann über HTTP an den Solr Server geschickt, ebenso wie die initialen Daten.

Und ja, es gibt auch schon was in Drupal: Robert Douglas baut für Acquia an einer Solr Implementierung. Acquia planen Solr/Lucene als Hosting Lösung anzubieten. D.h. Du installierst Dir das Modul auf Deinem Webspace bekommst von Acquia die Zugangsdaten zu "deiner" Suchmaschine, trägst die in Dein Modul ein, und der Rest kann Dir egal sein. Wie bezahlbar das allerdings ist weiß ich nicht.

Was ich auch nicht weiß, ist wie leicht man Lucene auf einem Shared Hosting Server ans Laufen bekommt.
