Open Source – Out Source

Ich bin ja immer noch schwer begeistert davon, dass Chrisitan mein Layout neu gemacht hat indem er mir einfach eine CSS Datei geschickt hat.

Konnexus frug mich, ob ich das jetzt outsourcen würde, worauf hin ich gesagt habe, dass ich es eher mit mir geschehen lassen würde, mit Begeisterung.

Also nicht, dass ich nicht gewußt hätte, das sowas geht. Aber die Leichtigkeit und die Großartigkeit des Ergebnisses hauen mich doch immer noch um. Wie schon erwähnt, ich hab keine Zeile HTML verändert – reiner CSS Relaunch. Sowas sollte man viel öfter machen. Kuhl wäre ja, wenn man statt Themes einfach nur noch CSS-Dateien auswählen würden. Die könnten ja sogar im Unterschied zu Themes auch von einem entfernten Rechner eingebunden werden. Einfach URL in einem Formular im Backend austauschen – fertig ist das neue Layout.

Alles was man für ein besseres Web bräuchte wären HTML, das ein wenig mehr standartisiert ist, was Klassen und IDs angeht und Leute, die sich trauen auch die CSSe anderer zu hosten. Was man dann alles duftes machen könnte …

Danke nochmal an Chrisitian, nicht nur für das CSS sondern für die schöne Idee.

P.S.: Ja, es ist viel zu heiß zum schlafen in dieser von Gott verlassenen Stadt, so dass ich noch blogge und schwitze.
