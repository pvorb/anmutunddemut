Battleground State Blogsphere - now updated!

As Bloggers are an important Group of potential Drupal Users the Blogsphere is a Battleground State for Drupal. Migration from market leader Wordpress to Drupal is extremely difficult. But even after a successfull migration, and the installation of usefull modules a common Blogger will miss keyfeatures and the usability know from Wordpress.

In Core but not activated on installation

  • Search
  • Trackback
  • BloggerAPI
  • Main Categories and Tags

Missing in Core but (mostly) available as Module

  • Archive
  • Spam-Protection (what a about false negative?)
  • WYSIWYG Editor
  • Moderation
  • Permlink (Path & Path Auto)

Missing Modules

  • Email-Notification on new Comments.
  • Predated publishing

Usability

  • Compex Node-Creation Form
  • Dashboard (???)

Architectural Aspects

  • Theming is mostly a job for developers.

Drupals Strengths and Advantages for Blogger

  • Easy way to create new node Types, but no Overviews just for Nodes
  • Easy (but hidden) Aggregator
  • Easy to add community features

Update: Learnings from the Session

Problem: Wordpress has more blogger-centered Plugin, that are earlier released as well.
Solution: We cant's help that.

Problem: Wordpress has a better Media Asset Management
Solution: Write a great Asset Management Module!

Problem: Wordpress has a Backend / Dashboard that is streamlined for the needs of Bloggers.
Solution: A Dashboard as an alterantive Drupals Backend is not that much work.
- It should be streamlined for Users, who are only Content-Creators.
- Overview of the Content (Numbers)
- Latest Comments
- Latest Drafts
- QuickPress
- An improved Navigation-Menu

Problem: Node Creation Form is generic but pretty difficult to use.
Solution: An improved Node-Creation Form can be mostly done with Theming.
- Strip away most of the blocks
- Add WYSIWYG
- Add Media-Management
- Add Buffer and Publish Buttons

English:
basicly drupal has everything to recreate a wordpress-like feeling. Only problem, this functions are not out-of-the-box.
