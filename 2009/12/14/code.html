co.de

Das ist auch neu. Eben kam per Post die schriftliche, auf totem Holz gedruckte Bestätigung, dass ich "mit anmutunddemut.de eine der wichtigsten Seiten im deutschen Markt betreibe". Der nette Martin Steinkamp war so freundlich, mir auf Grund dessen eine Einladung zur "Sunrise-Phase" von co.de zu schicken. Erst dachte ich ja das wäre sowas duftes wie Github … so von wegen Co.de = Code … ihr versteht schon … aber in der Tat glaubt der nette Martin Steinkamp lediglich, co.de parallel zu co.uk vermarkten zu können. Dabei habe ich heute erst erfahren, dass dass "co" dabei für "commercial" stehen soll. Auf Grund der allumfassenden Verwendung hatte ich immer gedacht, es würde für "commonwealth" … hihi … was für eine lustige Idee!
