Keine Atempause - Geschichte wird gemacht - Es geht voran!

Anne ist doch der beste Beste. Kein anderes Blog ist so informativ, wie das des jungen Holländers. Letzte Neuigkeit: die Firefox nigthly Builds [das sind die aktuellen Entwicklungsversionen einer Software] kommen jetzt mit nativen SVG-Support. Und die Firfox Version 1.1 soll dann damit ausgeliefert werden. Yeeehawww!

Die Qualität von Annes Blog läßt sich im Übrigen daran erkennen, dass Anne weiß und es auch schreibt, dass eine Implementierung von SVG zwar ein Schritt nach vorne ist, SVG selbst aber als ein Standard mit erheblichen Kinderkrankheiten bezeichnet werden muss. Nichts desto Trotz freue ich mich riesig auf den Firefox 1.1 und bin jetzt gerde sogar ein bischen elektrisiert. SVGwird Realität.

Ein weiteres gutes Argument, den IE von diesem Blog auszusperren, was im übrigen gut läuft: gestern haben über 200 Leute von mir erfahren, dass sie einen Scheißbrowser verwenden.
