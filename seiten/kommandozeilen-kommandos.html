Kommandozeilen Kommandos

In the beginning there was the commandline.
Es gibt eine Welt, in die man als PHP-Programmierer inzwischen nicht mehr sooft muss. Wenn man sich aber auf die Kommandozeile begibt, dann sollte man nicht vergessen ein paar gute Freunde mitzunhemen.

Vi

Datei am Ende Öffnen
vi datei.endung +
Seite runter / Seite hoch
ctrl f - wie forward
ctrl b - wie backward
Vorwärts suchen
/Suchbegriff
Rückwärtssuchen
?Suchbegriff
Weitersuchen
n (wie next)

My Top

mytop -h localhost -u USER -p PASSWORD -d ''

tail

tail -f datei.endung -n NummerDerZeilen

Zip

Einen Ordner rekursiv mit Passwortschutz zippen
zip -r -e [name-der-zip-datei] [name-des-ordners]

Tunneln

ssh -L2222:zielserver.domain.de:22 benutzer@tunnelserver.domain.de
Dann kann man über Port 2222 eine SFTP Verbindung zu localhost öffnen, mit dem Benutzernamen und dem Passwort, dass man auf dem Zielserver hat.

Apache

Apache Restart

netstat -a(n)t | sort | less (um zu sehen, mit wem alles (nicht mehr) gesprochen wird)
/etc/init.d/apache2 stop
netstat -ant | grep (-c) :80 (wiederholt, ich warte gerne bis alle Verbindungen wirklich weg vom Fenster sind, auch die FIN_WAIT-Reste)
/etc/init.d/apache2 start

Apache Benchmark

ab -c [Anzahl der simultanen Request] -n [Anzahl der Request]

Ordner und Dateien

Anzahl der Dateien in einem Ordner

ls -1 | wc -l
und rekrusiv
ls -R -1 | wc -l

Größe einen Verzeichnisses, rekursiv

du . -c -s -h

Dateien finden, die größer sind als 200MB

find / -type f -size +200000k -exec ls -lh {} \; | awk '{ print $9 ": " $5 }'

Symbolische Links anlegen

ln -s /ordner/nocheinordner/ziel name

grep

Findet eine Zeichenkette in allen Dateien des aktuellen Ordners
grep zeichenkette *

Alle Vorkommen von xyz in Datei.name zählen.
grep 'xyz' datei.name | wc -l

User

User anlegen

adduser Username
useradd -G Gruppe -p Passwort Username

Passwort eines Users ändern

passwd Username

User eine Gruppe hinzufügen

usermod -g www Username

SVN

Initialer Checkout

svn co http://svn.example.com/svn/mein-projekt

Ein Verzeichnis ins Repoistory einfügen

svn add name-des-verzeichnisses/

Ein Verzeichnis einchecken

svn ci name-des-verzeichnisses/ -m "Grund des Checkins"

Mein Verzeichnis aktualisieren

svn update

Git

Globales Setup

git config --global user.name "MeinName"
git config --global user.email meine-email@example.xom

Lokales Repository erstellen

mkdir mein-projekt
cd mein-projekt
git init

Initiale Synchronisation

git remote add origin git@github.com:mein-user-name-bei-github/mein-projekt.git
git push origin master

Ein Verzeichnis ins Repoistory einfügen

git add name-des-verzeichnisses/

Eine DAtei aus dem Repository entfernen

git rm name-der-datei

Ein ganzes Verzeichnis aus dem Repository entfernen

git rm -r name-des-verzeichnisses/

Ein Verzeichnis einchecken

git commit name-des-verzeichnisses/ -m "Grund des Checkins"

Lokales Reository mit dem Master synchronisieren

git push origin master oder
git push

Einen entfernten Fork hinzufügen

git remote add namedesforks git://github.com/user/projekt.git

Einen entfernten Fork ziehen

git fetch namedesforks

Zwei Repos mergen

git merge namedesforks/namedesmasters

Einen entfernten Fork ziehen (fetch und merge)

git pull namedesforks namedesrepositories

Einen Branch erstellen

git checkout -b namedesbranches

Sich von Branch zu Branch schwingen

git checkout namedesbranches

Alle Branches lokalen anzeigen

git branch

Alle Branches, auch remote anzeigen

git branch -a

Konsole auf dem Mac

Shortcuts einrichten

vi .bash_profile
alias shortcut='long command which does - something'

Linux Version herausfinden

cat /etc/apt/sources.list
